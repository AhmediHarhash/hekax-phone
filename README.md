# HEKAX Phone

## AI Communication Platform

HEKAX Phone is an enterprise communication operating system combining voice automation, multi-channel messaging, and AI-driven decisioning. It is designed for service-driven teams that require reliable intake, routing, follow-up, and operational oversight.

---

## System Overview

HEKAX Phone is composed of three primary worlds:

### 1. Calls World

Handles inbound and outbound calls with AI-driven analysis, transcription, and routing.

**Capabilities:**
- AI receptionist
- Call intent detection
- Emergency classification
- Industry-specific intake flows
- Real-time transcription

### 2. Messages World

Unified messaging OS for all text channels.

**Capabilities:**
- SMS, WhatsApp, Facebook, Telegram, Email
- Smart Inbox with prioritization
- AI drafting and automated responses
- Sentiment and intent scoring
- Conversation assignment and state handling

### 3. Live Control World

Real-time operational monitoring.

**Capabilities:**
- SLA status
- Queue health
- Agent presence
- Performance indicators
- Decision logs

---

## Core Components

### AI Orchestrator

Central routing mechanism for:
- Intent detection
- Smart replies
- Summaries
- Lead scoring
- Sentiment analysis

### Codex Brain

Applies multi-layer policies to classify events and determine actions.

### Event Mesh

Distributed event processing pipeline using Redis Streams for scalable, real-time communication.

### Contact Graph

Identity and relationship resolution across channels.

### State Machine Layer

Formal workflow engine for leads, conversations, and custom processes.

## Industry Alignment

HEKAX Phone includes pre-configured intelligence for:

- Legal firms
- Medical groups
- Home services (HVAC, plumbing, electrical)
- SaaS companies
- Call centers and BPO
- Generic SMB

Each vertical includes custom triage logic, labeling, and Cockpit dashboards.
